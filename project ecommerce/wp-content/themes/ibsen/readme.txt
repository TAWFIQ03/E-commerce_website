== Installation ==
1. In your WordPress Dashboard, nagivate to **Appearance > Themes** and click the **Add New** button.
2. Type **Ibsen** in the search form and press the **Enter** key on your keyboard.
3. Click the **Activate** button to begin using Ibsen on your website.

== Customization ==
1. Navigate to **Appearance > Customize**.
2. Customize the style of your website by adding a logo, setting your preferred layout, changing the fonts and selecting a custom accent color.

== Screenshot Content ==
1. Navigate to **Pages > Add New**.
2. Add a **Cover** block, upload or select an image, select **Full width** alignment option.
3. Below the Cover block, add a **Columns** block, select **Two columns; one-third, two-thirds split** variation.
3b. In the first column, add a WooCommerce **Featured Category** block, write **equal-height** CSS class in **Advanced** option.
3c. In the second column, add a WooCommerce **Newest Products** block, select **3 columns** and **2 rows** layout.
4. In **Document > Page Attributes**, select **Transparent Header** template.
5. Click **Publish**.
